# dxm\_extract\.py \- 冬小麦种植面积调整工具

## 概述

该 Python 脚本用于调整冬小麦（dxm）矢量数据，核心目标是使每个行政村范围内的冬小麦面积与目标面积字段（种植面）在容差范围内匹配。脚本结合遥感影像分析（NDVI 计算、随机森林分类）和面向对象的斑块选择策略，实现基于像素概率的冬小麦区域筛选与面积校准，最终输出调整后的矢量数据及行政村级面积报告。

## 功能特点

1. **多源数据支持**：兼容 TIFF/IMG 等栅格影像、SHP/GPKG/GeoJSON 等矢量数据；

2. **智能分类策略**：

    - 若存在参考冬小麦矢量，训练随机森林模型（以栅格波段 \+ NDVI 为特征）；

    - 无参考矢量时，采用 NDVI 阈值法作为备选；

3. **云检测过滤**：基于可见光波段亮度和 NDVI 值识别云像素，排除干扰；

4. **面向对象的斑块选择**：按概率评分筛选斑块，支持部分斑块裁剪以精准匹配目标面积；

5. **后处理优化**：移除细碎噪点、修复拓扑错误、去重，保证矢量数据质量；

6. **多维度报告**：输出行政村级目标面积 / 实际面积 / 差值，高亮超容差结果。

## 数据准备

### 目录结构

脚本需按以下目录结构放置数据（根目录为脚本所在目录的上级目录）：

```Plain Text
├── dxm_extract.py          # 主脚本
├── image/         # 遥感影像目录（必填）
│   └── *.tif/*.tiff/*.img  # 多波段遥感影像（至少3波段，4+波段可计算NDVI）
├── border/                 # 行政村边界矢量目录（必填）
│   └── *.shp/*.geojson/*.gpkg  # 包含目标面积字段的行政村矢量
└── dxm/                    # 参考冬小麦矢量目录（可选）
    └── *.shp/*.geojson/*.gpkg  # 用于训练的冬小麦参考矢量
```

### 数据要求

1. **遥感影像**：

    - 投影坐标系（建议 UTM/WGS84 Web Mercator）；

    - 波段顺序：默认 Band3 = 红波段、Band4 = 近红外波段（用于 NDVI 计算）；

2. **行政村矢量**：

    - 包含目标面积字段（优先识别：种植面 /target/plant\_area/area/ 种植面积）；

    - 若无目标面积字段，将使用行政村多边形面积作为替代（会触发警告）；

3. **参考冬小麦矢量（可选）**：

    - 与遥感影像投影一致（脚本会自动重投影）；

    - 几何类型为 Polygon/MultiPolygon，无拓扑错误。

## 使用方法

### 基础运行

```bash
# 进入脚本所在目录
cd /path/to/script_dir
# 执行脚本
python dxm_extract.py
```

### 关键参数（脚本内可调整）

|参数名|位置|默认值|说明|
|---|---|---|---|
|`thresh`|行政村斑块筛选段|0\.1|初始概率阈值，用于筛选候选斑块|
|`high_conf_threshold`|高分斑块判定|0\.75|高置信度斑块的概率阈值（触发加分）|
|`area_tolerance`|面积容差|1000 m²|允许的面积超调软上限|
|`min_obj_m2`|后处理噪点移除|100 m²|移除小于该面积的细碎斑块|
|`sigma`|概率图平滑|1\.0|高斯平滑核大小，增强空间连续性|

## 输出结果

### 文件输出

1. `workspace/adjusted_dxm.gpkg`：调整后的冬小麦矢量数据，包含字段：

    - `village_idx`：行政村索引；

    - `area_m2`：冬小麦斑块面积（平方米）；

    - `geometry`：斑块几何信息；

2. `workspace/adjust_report.csv`：行政村级面积报告，字段：

    - `village_idx`：行政村索引；

    - `target_area_m2`：目标面积（平方米）；

    - `achieved_area_m2`：调整后实际面积（平方米）；

    - `diff_m2`：面积差值（实际 \- 目标）。

### 控制台输出

- 数据加载日志（栅格 / 矢量路径、CRS、波段数）；

- 关键步骤日志（NDVI 计算、云检测、模型训练、斑块筛选）；

- 警告信息（无目标面积字段、面积差值超 3000m² 的行政村）；

- 输出文件路径。

## 核心流程说明

1. **数据加载与预处理**：

    - 自动识别并加载栅格 / 矢量数据；

    - 统一投影坐标系（以栅格 CRS 为准）；

    - 计算 NDVI（4 \+ 波段）、云掩码；

2. **概率图生成**：

    - 有参考矢量：训练随机森林模型，输出冬小麦概率图；

    - 无参考矢量：NDVI 阈值归一化生成概率图；

3. **行政村级斑块筛选**：

    - 按概率评分排序斑块；

    - 累加斑块面积至目标值（支持部分斑块裁剪）；

    - 不足目标时补充高概率像素；

4. **后处理与矢量化**：

    - 移除细碎噪点、修复拓扑；

    - 栅格转矢量并按行政村分割；

    - 去重、计算面积、保存结果。

## 常见问题

1. **脚本提示 “Required data not found”**：

    - 检查`xiangfen_image/`和`border/`目录是否存在，且包含对应格式的数据；

2. **NDVI 计算失败**：

    - 确认遥感影像波段数≥4，且 Band3 = 红波段、Band4 = 近红外波段；

3. **随机森林训练跳过**：

    - 参考矢量像素数不足（需≥100 有效像素），自动切换为 NDVI 阈值法；

4. **输出矢量无数据**：

    - 检查概率图是否有效（云覆盖过高、NDVI 阈值设置不合理）；

    - 确认行政村矢量与栅格投影匹配。

## 注意事项

1. 遥感影像建议使用经过大气校正的表面反射率数据，提升 NDVI 准确性；

2. 目标面积字段单位需为 “平方米”，否则会导致面积匹配偏差；

3. 脚本处理大尺寸栅格时（如≥1GB），建议增加内存（≥16GB），避免内存溢出；

4. 输出矢量的投影与输入栅格一致，若需地理坐标系，可手动转换（如 EPSG:4326）。
